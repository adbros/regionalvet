== Description ==

= An easy way to get a floating div. =

Thanks to Floating Div, you’ll have the opportunity to add some movement in your website.

You’ll be able to continuously display a div, like a menu, that stick when you’re scrolling.

The goal of the plugin is to obtain a permanent div, that will stay displayed on screen.

= How to use our plugin ? =

* You just have to download the plugin from the Wordpress Repo.
* Activate your plugin on your Dashboard.
* **Install and Activate the CMB2 framework**, it will be prompted in your plugin tab. (See below for more info)
* At this point you will see a Floating Div tabs, you just have to create a new Floating Div by completing the fields.
* Once created, go back to the list of your Floating Div and you will see a Shortcode.
* All you have to do, is put the Shortcode wherever you want to get your Floating Div.

The CMB2 plugin is used to handle the data between the admin panel and the front-end (the "user" view).
Nothing will change on your website, it is only required to make your settings with our plugins easier.

= Here is an example of the possible uses : =

* You can keep a menu for the user.
* Keep on screen the amount of the current cart.
* Show some important information, like a new release.
* Using the plugin like a top bar.
* Permanently display social link, like Twitter or Facebook.
